# 集合(无序, 元素不重复)
	创建, 特性, 常用方法
	s = {1, 2, 3, 1, 2, 3}  ---> s = {1, 2, 3}
	s = {}			---> 是一个字典
	s = set()		---> 创建空集合
	s = {1}			---> 集合
	t = (1)			---> 整形， 不是元组
	t = (1, )		---> 元组
	s = {1, 2, 3, [1, 2, 3]} ---> 集合里面的元素必须是不可变的(不能增删改)

	1 in s 			---> 集合支持成员操作符

	增， 删， 查(交集， 并集， 差集， 对等差分， 判断是否为父集/子集， 是否有交集)


	应用场景: 
		1). 列表去重
		2). 交集， 并集， 差集
